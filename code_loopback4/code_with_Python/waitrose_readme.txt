quick guide for developers 
1. Install Nodejs (10.5)
2. Install NPM (6.9)
3. Install Python (3.7)
4. Install PIP
5. Install Python Flask
6. npm install loopback
7. npm install loopback context

Start nodejs server .
1 . Navigate to inventory folder  in cmd prompt
2. Execute below command 
npm start

3. Go to python script folder 
4. execute 

getInventory_v2.py




you will be able to see the api explorer at http:// 127.0.0.1:3001


Please use postman to make get request with auth token.
